# tslint-config-beanloop
An opinionated tslint config.

## Installation

```sh
yarn add tslint-config-beanloop
npm install --save tslint-config-beanloop
```

## Usage

Set your tslint config to:

```json
{
  "extends": [
    "tslint-config-beanloop"
  ]
}
```
